# Sample configuration for generals-bot.
#
# Usage:
# 1. Copy this file to a file named run_config.txt.
# 2. Put that run_config.txt in the folder that CONTAINS the repository folder.
#    Example:
#      Repository:   C:\Bots\generals-bot
#      Real config:  C:\Bots\run_config.txt
# 3. Update the paths below for your machine.
# 4. Lines starting with # are ignored.
#
# Notes:
# - Most values are filesystem paths.
# - repo_root should point at the repository folder itself.
# - log_folder and grouped_folder are required by active logging / analysis flows.
# - current_gen_python_path is the Python executable used for the repo's current bot code.
# - legacy_python_path is the Python executable used for external or older bot repos.
# - The *_pos values are used by the viewer window-position loader.
# - bot_logs_root is optional in practice because some scripts have a fallback,
#   but defining it explicitly is recommended.
#
